# TinyLearners 🌈

TinyLearners is a colorful MERN stack educational web application designed for toddlers and young children. The platform provides interactive learning experiences through, educational categories, and a parent-friendly dashboard.

The goal of this project is to make early childhood learning fun, engaging, and visually interactive.

---

# ✨ Features

* 🔤 Alphabet Learning Flashcards
* 🔢 Number Learning
* 🦁 Animal Learning Section
* 🍎 Fruits Learning Section
* 🎨 Colors Learning Section
* 🔺 Shapes Learning Section
* 📊 Parent Dashboard
* 🌈 Animated and Kid-Friendly UI
* 📱 Responsive Design
* ☁️ MongoDB Atlas Database Integration
* ⚡ REST API Backend with Express.js

---

# 🛠️ Tech Stack

## Frontend

* React.js
* Vite
* Tailwind CSS
* Framer Motion
* React Router DOM

## Backend

* Node.js
* Express.js

## Database

* MongoDB Atlas

# 🚀 Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone <your-github-repo-link>
```

---

## 2️⃣ Install Frontend Dependencies

```bash
npm install
```

---

## 3️⃣ Install Backend Dependencies

```bash
cd backend
npm install
```

---

# 🔐 Environment Variables

Create a `.env` file inside the backend folder and add:

```env
MONGO_URI=your_mongodb_atlas_connection_string
PORT=5000
```

---

# ▶️ Running the Project

## Run Frontend

```bash
npm run dev
```

## Run Backend

```bash
cd backend
node server.js
```
